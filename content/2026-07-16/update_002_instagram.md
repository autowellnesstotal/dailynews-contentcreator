### Slide 3 — ¿Qué es la memoria persistente?
**Título:** Claude Code ahora recuerda

**Cuerpo:**
Anthropic acaba de lanzar dos sistemas de memoria en Claude Code:

🧠 **Auto Memory** — guarda automáticamente el contexto de tu proyecto entre sesiones. Sin que hagas nada.

📄 **CLAUDE.md** — un archivo de instrucciones permanentes que Claude lee al inicio de cada sesión.

Resultado: Claude ya sabe quién eres y cómo trabajas desde el primer mensaje.

### Slide 4 — Auto Memory: sin configuración
**Título:** Auto Memory — cómo funciona

**Cuerpo:**
Claude Code observa tus conversaciones y guarda automáticamente:

✅ El contexto de tu proyecto o negocio
✅ Tu estilo de comunicación preferido
✅ Notas de errores que ya resolviste
✅ Patrones de trabajo que usas con frecuencia

**No tienes que configurar nada.** Solo trabajar normalmente.

---

### Slide 5 — CLAUDE.md: tus instrucciones permanentes
**Título:** CLAUDE.md — el archivo que lo cambia todo

**Cuerpo:**
¿Tienes instrucciones que quieres que Claude siga SIEMPRE?

Escríbelas una vez en un archivo llamado CLAUDE.md y Claude las leerá automáticamente en cada sesión.

Ejemplo de lo que puedes escribir:
*"Soy dueño de una cafetería en Bogotá. Siempre respóndeme en español. Cuando crees contenido, hazlo para Instagram."*

Una vez configurado — nunca más lo repites.
